256bit visualization

Did you see phrases such as "256 bit encryption" or "256 bit security" somewhere?
But how big is 256 bits really? This demo will try to give you some idea how long it takes to count to 2^256.
Imagine, hacker can check one number per CPU cycle. How long would it take to go through all 2^256 combinations?
Select different starting points and different speeds to see how long it would take to check every single possible combination.
